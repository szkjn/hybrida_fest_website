# hybrida_fest_website

# clone project

```bash
git clone https://github.com/szkjn/hybrida_fest_website.git
```

# install

```bash
cd app
npm i
```

## start the server and open the app in a new browser tab

```bash
npm run dev -- --open
```

## building for production

```bash
npm run build
```

## preview the production build

```bash
npm run preview
```
